## File Changes
- Always ask for my confirmation before creating, editing, or deleting any file.
- Show me a diff of the proposed changes and wait for my explicit approval before applying.
- Never apply multiple file changes at once without confirming each one.

## General Behavior
- Explain what you plan to do before doing it.
- If unsure about intent, ask for clarification instead of assuming.
- Prefer small, focused changes over large rewrites.

## Restrictions
- Do not run destructive commands (rm, git reset --hard, etc.) without explicit confirmation.
- Do not push to any remote git repository without asking first.
